cBio experiments

This is a blank page that loads the cbio.js script. All the action is in the browser's command line.
To load the same library in your webApp just include this line (there are NO dependencies from other libraries, such as jQuery):

<script src="https://ibl.github.io/cbio/cbio.js"></script>

For more informaion and versioned source code see https://github.com/ibl/cbio.
